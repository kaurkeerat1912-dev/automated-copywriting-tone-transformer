# Automated Copywriting & Tone Transformer AI

## Overview
An AI-powered marketing copy generator built using Python, Streamlit, and Google's Gemini API.

## Features
- Generate marketing copy for Instagram, LinkedIn, and Email
- Multiple writing tones:
  - Professional
  - Friendly
  - Luxury
  - Casual
  - Witty
- Adjustable creativity using Temperature slider
- Powered by Google Gemini API

## Tech Stack
- Python
- Streamlit
- Google Gemini API
- Prompt Engineering
- Git & GitHub
